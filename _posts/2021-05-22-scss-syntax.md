## 주석 comment

기본 css에서 사용하던 방식 + js 주석 처럼 `//` 슬래시 두 개도 사용 가능하다.

```scss
// 컴파일 안 됨

// 한 줄 주석

/* 컴파일 됨 */


/* 여러
줄
    주석
*/
```
